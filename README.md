# streamsets-haversine-processor

This was a quick & dirty hack for a demo. It calculates the distance, in kilometers, between two points.

input fields:
* `/latitude`
* `/longitude`
* `/previous.latitude`
* `/previous.latitude`

output field:
* `/distance`
